Projetos

Classificador de imagens de roupas

Este projeto envolve um modelo de classificação de imagens que identifica tipos de roupas, utilizando Python e redes neurais convolucionais (CNNs). A ideia foi aplicar CNNs para treinar o modelo a reconhecer e diferenciar peças com base nas características visuais. Com isso, o modelo processa e classifica cada imagem, ajudando a entender a categoria de cada roupa de forma automática e precisa.

Classificador de resenhas do IMDB

Neste projeto, construí um modelo para classificar resenhas do IMDB, identificando se são positivas ou negativas. Usando Python e técnicas de Processamento de Linguagem Natural (NLP), o modelo analisa o texto das resenhas e aplica regressões para prever a polaridade do sentimento. A proposta é automatizar a análise de sentimento, ajudando a interpretar a recepção dos filmes de forma mais objetiva e rápida.

Modelo de predição de CPOs

Desenvolvi um modelo de regressão para prever o preço ideal por pedido, ajustado ao comportamento passado dos usuários, à audiência e à vertical. Com Python, técnicas de otimização e machine learning, o modelo reduziu os custos em 37%, mantendo uma perda mínima de apenas 4% na conversão. Esse equilíbrio trouxe mais eficiência para as campanhas do time de CRM.
